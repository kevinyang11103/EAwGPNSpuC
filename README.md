## 项目概述


## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端**: Vue.js

## 功能模块

- 用户兴趣深度挖掘
- 精准书籍推荐
- 多维度推荐策略（热门书籍、新书速递、个性化需求）

## 系统角色

- 用户：获取个性化书籍推荐
- 系统管理员：维护系统运行、管理用户数据

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Vue.js 2.x

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 前端Vue.js项目部署
5. 系统测试与运行
